What is STim ?

STim is an online game store that provides third party games from different publishers. It provides user with updated news of game updates, on-going event, and game exhibition launch. To support the publisher, all games must be purchased. As a web designer, you are assigned to design an attractive website to guide new user to use the website as their first time using an online game store to searching new games, purchase a game, and following an event.

Our History

STim beridri pada tahun 2021. dimulai dengan 2 orang pemuda yang suka menghabiskan waktunya dengan cara bermain game, dan memiliki ambisi untuk membuat platform milik mereka sendiri agar orang-orang dapat bermain melalui platform yang mereka buat, dan akhirnya dengan kerja keras. mereka dapat membangun STim yang kini sudah menjadi sebuah platform penyedia game yang dapat dinikmati oleh semua orang
